Option Description How to use
Remove styling You can remove bullets or numbers from a list. Add the class list-unstyled
Spacing You can additional spacing between list items to create additional separation. Add the class lst-spcd
Remove bullets in nested list You can remove the bullet or number from a nested list, while maintaining the identation Add the class lst-none
Lower alpha You can apply lower case alphabet letters (a, b, c...) to a an ordered list. Use mainly as a nested second level list. Add the class lst-lwr-alph
Upper alpha You can apply upper case alphabet letters (A, B, C...) to a an ordered list. Use mainly in legal documents. Add the class lst-upr-alph
Lower Roman You can apply lower roman numeral (i, ii, iii...) to a an ordered list. Use mainly in legal documents. Add the class lst-lwr-rmn
Upper Roman You can apply upper roman numeral (I, II, III...) to a an ordered list. Use mainly in legal documents. Add the class lst-upr-rmn
